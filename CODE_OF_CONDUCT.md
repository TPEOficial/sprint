# Code of Conduct

## Our Commitment

Sprint is committed to providing a welcoming, inclusive, and respectful environment for everyone.

We value:

- Professional collaboration
- Constructive feedback
- Technical excellence
- Respectful communication

---

## Expected Behavior

Participants should:

- Use respectful and inclusive language.
- Be open to constructive criticism.
- Focus on technical discussions.
- Respect differing viewpoints and experience levels.
- Prioritize collaboration over conflict.

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discriminatory language.
- Personal attacks.
- Trolling or disruptive behavior.
- Sharing private information without consent.
- Any form of intimidation.

---

## Enforcement

Project maintainers may:

- Request changes.
- Remove contributions that violate guidelines.
- Restrict participation in severe cases.

---

## Scope

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Any official Sprint communication channels

---

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the maintainers.

All reports will be handled with discretion and seriousness.

---

Thank you for helping keep Sprint a professional and collaborative project.
